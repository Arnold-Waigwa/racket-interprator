# Overview

This project is a custom interpreter written in **Racket** that implements a simplified Scheme-like language.

It supports evaluation of expressions, environments, variable binding, conditionals, function application, and a REPL.

The goal of this project was to understand how programming languages work internally by building:

- an evaluator (`i-eval`)
- an environment model (frames + bindings)
- a procedure application system
- a simple REPL
